

import.meta
컨텍스트별 메타데이터를 JavaScript 모듈에 노출하는 객체입니다.


import.meta는 현재 실행 중인 ES 모듈의 메타데이터(정보) 를 제공하는 JavaScript 내장 객체입니다.
주로 모듈의 절대 경로(url)를 확인하거나, Vite 같은 빌드 도구에서 환경 변수를 가져올 때 사용합니다.
가장 대표적인 활용 사례는 다음과 같습니다.
현재 모듈의 절대 경로 확인 (import.meta.url): 모듈 파일이 위치한 정확한 URL을 문자열로 반환합니다.
환경 변수 접근 (import.meta.env): Vite, Nuxt 등에서 .env 파일에 정의된 환경 변수를 읽어올 때 사용합니다.
사용 예시는 아래와 같습니다.javascript// 
모듈의 경로 확인
console.log(import.meta.url); 
// 출력 예: "file:///C:/project/src/main.js"

// Vite 환경에서 환경 변수 확인
if (import.meta.env.DEV) {
  console.log("개발 모드입니다.");
}

super
이 super키워드는 부모 객체의 생성자를 호출하거나 부모 객체의 속성에 접근할 수 있도록 합니다.

import()
이 import()구문을 사용하면 모듈을 비모듈 환경일 수도 있는 곳에 비동기적으로 동적으로 로드할 수 있습니다.
