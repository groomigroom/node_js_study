

await
비동기 함수를 일시 중지하고 다시 시작하며, 프로미스의 완료/거부를 기다립니다.

자바스크립트에서 await는 Promise가 처리(성공 또는 실패)될 때까지 기다리게 하는 키워드입니다.
이를 통해 비동기 작업을 마치 동기 작업처럼 순차적이고 직관적으로 작성할 수 있습니다.
주요 특징과 동작 방식은 다음과 같습니다.
1. 주요 기능
실행 일시 중지 및 재개: await를 만나면 해당 비동기 작업이 끝날 때까지 기다린 후(대기), 
완료되면 결과값을 반환하고 다음 코드를 실행합니다.
스레드 차단 안 함: 기다리는 동안 자바스크립트 메인 스레드는 멈추지 않고 다른 작업을 계속 수행하므로(논블로킹), 
프로그램이 멈추지 않습니다.
2. 필수 조건
await는 반드시 async 키워드가 붙은 함수 내부에서만 사용할 수 있습니다.
보통 서버에서 데이터를 받아오거나(API 호출), 파일 읽기 등 시간이 걸리는 작업 앞에 사용합니다.
3. 간단한 예제 코드javascript
async function fetchUserData() {
  // 1. 서버에 요청을 보내고 응답이 올 때까지 대기
  const response = await fetch('https://example.com');
  
  // 2. 응답 데이터를 JSON 형태로 변환할 때까지 대기
  const user = await response.json();
  
  // 3. 변환된 데이터를 바탕으로 출력
  console.log(user.name);
}
4. 에러 처리await로 기다리던 Promise가 실패(거부)할 경우 에러가 발생하므로,
이를 처리하려면 try...catch 문을 함께 사용해야 안전합니다.javascript
async function getData() {
  try {
    const data = await fetchApi();
    console.log(data);
  } catch (error) {
    console.error('에러 발생:', error);
  }
}
